Auto Web Login Bot
============================================================
## TOOLS REQUIRED ##
1. New version of Python 2
2. Google Chrome Browser
3. Selenium browser automation toolkit
4. Chrome Driver - web driver for chrome

## PROCEDURE ##
1. Install PIP
2. Install selenium using command : pip install selenium

3. Download the Selenium Chrome Driver executable, extract the chromedriver.exe executable, and place the executable in any directory
4. Restart your computer
